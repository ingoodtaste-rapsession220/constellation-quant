## 🚀 Setting up the software

Once the application opens, it will perform a quick check of your system. You do not need to configure any hidden files or complex parameters.

## 📊 Understanding the results

The main dashboard shows a ranked list of stocks. Each stock receives a score based on its recent temporal movement and its relationship to other assets in the index.

* Rank: The position of the stock as determined by the model.
* Ticker: The standard symbol used to identify the company.
* Score: A numerical representation of the predicted strength.
* Trend: An indicator of recent price behavior.

You can sort these columns by clicking on the headers. This helps you narrow your focus to specific sectors or rankings.

## 🛠 Troubleshooting common issues

If the software fails to open or stops responding, follow these steps to resolve the issue.

* Check your network settings: The software requires an active internet connection to download market data. If your firewall blocks traffic, see if you can add an exception for the constellation-quant application.
* Update your system: Ensure you have the latest Windows updates installed. Older versions of Windows may lack the necessary components this software needs to run.
* Verify storage space: If the application crashes during the data download, you might need more disk space on your primary drive.
* Restart the application: Sometimes a simple restart clears temporary errors that prevent the tool from querying the server.

If the problem persists, ensure no other heavy programs are running in the background. Financial data processing uses system resources. Closing memory-intensive web browsers or other apps will make the ranking process faster.
